# costos_ingresos_demo
Análisis de viabilidad financiera utilizando Power BI. El proyecto implementa un esquema de estrella relacionando tablas de hechos (Ingresos/Costos) con dimensiones temporales.
Título del Proyecto


Contexto y Objetivo

Este proyecto se enfoca envisualizar el desempeño financiero de múltiples proyectos. El objetivo principal es monitorear la salud económica comparando los Costos Totales frente a los Ingresos Reales.



Herramientas y Metodología

Power BI: Para la ingesta de datos, modelado y visualización interactiva.
DAX: Implementación de medidas agregadas como SUMX para costos totales y cálculos estadísticos (Promedio, Mediana y Varianza de Ingresos).
Modelado de Datos: Esquema de estrella conectando tablas de hechos (Costos, Ingresos) con dimensiones clave (Calendario, Descripción_proyecto, Disponibilidad).
